# vue/html-closing-bracket-spacing

标签闭合括号前必须有空格

### 为什么

保持一致的空格风格,提升代码可维护性。

### 建议

在自闭合标签的闭合括号前都添加一个空格。

### 错误示例

```vue
<template>
  <div/>
</template>
```

### 正确示例

```vue
<template>
  <div />
</template>
```
