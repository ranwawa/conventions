# vue/no-deprecated-vue-config-keycodes

禁止使用已废弃的`Vue.config.keyCodes`

### 为什么

在Vue.js 3.0.0+中，`Vue.config.keyCodes`已被废弃。

### 建议

不要使用`Vue.config.keyCodes`。

### 错误示例

```js
Vue.config.keyCodes = {
  // ...
};
```
