# react/jsx-pascal-case

必须使用驼峰命名自定义组件

### 为什么

HTML标签是全小写的,用于和HTML标签区分开来。

### 建议

请使用首字母大写的驼峰命名法）来命名。

### 错误示例

```jsx
<Test_component />
```

### 正确示例

```jsx
<TestComponent />
```
